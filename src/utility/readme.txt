This directory contains the utility files which help in the experimental analysis.

> constants.py: This file has all the MACROs used throughout the code.

> dcp_utilities.py: This file contains the utility functions for data cleaning
    and preprocessing.

> exp_utilities.py: This file contains the utility functions for general
    experimental assitance.

> gla_utilities.py: This file contains the utility functions for the group level
    analysis, e.g. group level GLM and statistical analysis.

> log.py: This file implements the logging.
